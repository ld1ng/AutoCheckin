## 使用

出于定制化而做的小小修改，只图一乐，感兴趣可以继续修改完善。

配置文件填写一卡通卡号、密码、邮箱，若不需要邮件通知可自行关闭。

start.sh填写自己的文件路径，./运行即可✈。
